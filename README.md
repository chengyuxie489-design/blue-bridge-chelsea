# Blue Bridge

An unofficial Chelsea supporters homepage built as a static, magazine-style fan site.

## Live Site

https://chengyuxie489-design.github.io/

## Features

- Art-led Chelsea homepage hero using a local visual asset
- Verified matchday information and club news links
- 2025/26 squad-number directory with position filters
- Match intelligence, tactical notes, chants, club identity, and source sections
- Responsive layout for desktop and mobile

## Local Preview

```powershell
python -m http.server 4173 --bind 127.0.0.1
```

Then open `http://127.0.0.1:4173/`.

## Notes

This is an unofficial fan project and is not affiliated with Chelsea Football Club.
